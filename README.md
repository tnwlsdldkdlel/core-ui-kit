# CoreUI Kit

재사용 가능한 UI 컴포넌트 라이브러리

## 기술 스택

- **React** - 컴포넌트 기반 UI 개발
- **TypeScript** - 타입 안전성
- **Tailwind CSS** - 유틸리티 우선 스타일링
- **Storybook** - 컴포넌트 문서화 및 개발
- **Vitest** - 테스트 프레임워크
- **React Testing Library** - 컴포넌트 테스트

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### Storybook 실행

```bash
npm run storybook
```

### 테스트 실행

```bash
npm test
```

### 빌드

```bash
npm run build
```

## 컴포넌트 목록

- Button
- Input
- Dropdown
- Modal
- Tabs
- Alert
- Toast
- Badge

## 프로젝트 구조

```
src/
  components/     # UI 컴포넌트
  stories/        # Storybook 스토리
  utils/          # 유틸리티 함수
  test/           # 테스트 설정
```
